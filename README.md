
#  Make It Right

A physics-based puzzle game where you draw shapes to guide a ball into the target bucket.
Use ramps, weights, and bridges across 15 increasingly challenging levels — but watch out
for red zones where drawing is forbidden!

Built with HTML, CSS, JavaScript, and Matter.js physics engine.

##  Setup Instructions
1. Clone this repository
2. Open the project folder in VS Code
3. Right-click `index.html` → Click **"Open with Live Server"**
4. The game will launch in your browser automatically

---

##  Controls
| Control | Action |
|---------|--------|
| Left Click + Drag | Draw a path or shape |
| Freehand (bottom bar) | Draw a custom freehand line |
| Rectangle (bottom bar) | Place a rectangle shape |
| Circle (bottom bar) | Place a circle shape |
| Plank (bottom bar) | Place a plank |
| Launch (top right) | Start the ball rolling |
| Undo | Remove the last drawn shape |
| Clear | Remove all drawn shapes |
